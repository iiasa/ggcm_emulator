The files
  'agmerra-tavg-avg-1980-2010-05deg-adjlon.nc4'
and
  'agmerra-prate-avg-1980-2010-05deg-adjlon.nc4'
contained in this folder were derived from the AgMERRA Climate Forcing
Dataset for Agricultural Modeling
(https://data.giss.nasa.gov/impacts/agmipcf/agmerra/) available
from NASA and described by Ruane et al., 2015
(https://doi.org/10.1016/j.agrformet.2014.09.016).

For more details, please consult with the documentation contained in the
file 'ggcm_emulator.py'.
